# JobHireGround UI Guidelines

## Design Philosophy

Professional SaaS first.
Gamified experience second.

Users should trust the platform as a hiring tool.

---

## Inspiration

* LinkedIn
* Indeed
* Modern SaaS dashboards
* Strategy game maps

Not inspired by:

* Pokémon GO
* MMORPG interfaces
* Anime game UIs

---

## Map Style

* Custom illustrated map
* SVG based
* Fast loading
* Simple roads
* Rivers
* City hubs
* Modern company buildings

Avoid:

* Dense forests
* Large fantasy environments
* Real-world GPS maps

---

## Markers

User:

* Circular profile icon

Company:

* Modern office building marker

Job:

* Opportunity marker

---

## Colors

Primary:

* Purple

Secondary:

* Blue

Background:

* Light neutral colors

---

## UX Principles

* Mobile-first
* Fast interactions
* Minimal clicks
* Accessible
* Clean spacing

---

## Performance Goals

* Fast initial load
* Lazy-loaded components
* SVG graphics
* Optimized images

Target:

* Smooth experience on low-end devices
